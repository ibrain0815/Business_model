# 공간 메모 + 물건 찾기 기능 기획서

> AR Agent Lab MVP 핵심 기능 상세 기획

---

## 1. 기능 개요

### 1.1 기능명

**"스페이스 메모리" (Space Memory)**

부제: "당신의 공간이 기억합니다"

### 1.2 핵심 가치 제안

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   "연간 60시간을 물건 찾는 데 낭비하지 마세요"              │
│                                                             │
│   ✓ 태그 없이 모든 물건의 마지막 위치 기억                 │
│   ✓ 공간에 고정된 메모로 잊지 않는 습관                    │
│   ✓ 나가기 전 자동 체크리스트 알림                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 문제-솔루션 매칭

| 사용자 문제 | 기존 해결책 | Space Memory 솔루션 |
|-----------|-----------|-------------------|
| 열쇠 어디 뒀지? | AirTag 부착 (개당 4만원) | AI가 마지막 위치 기억 (무료) |
| 우유 사야 하는데 매번 까먹어 | 메모앱 (확인 안 함) | 냉장고 앞 지나가면 AR 알림 |
| 나갈 때 뭐 챙겨야 하지? | 체크리스트 (수동) | 현관에서 자동 AR 체크리스트 |
| 택배 어디 뒀더라? | 기억에 의존 | "이 방에서 택배 어디?" 음성 질문 |

## 2. 상세 기능 설계

### 2.1 핵심 기능 3가지

#### 기능 A: 물건 위치 기억 (Object Memory)

**작동 방식**:
1. 사용자가 물건을 내려놓을 때 카메라가 자동 인식
2. AI가 물건 종류와 위치를 공간 맵에 저장
3. "열쇠 어디 있어?"라고 물으면 마지막 위치 AR 표시

**기술 요소**:
- SLAM (공간 매핑)
- Object Detection (물건 인식)
- Spatial Anchoring (위치 고정)

**사용자 시나리오**:
```
[시나리오 1: 열쇠 찾기]

1. 아침 출근 준비 중 열쇠가 안 보임
2. "열쇠 어디 있어?" 음성 명령
3. AR 화면에 거실 소파 옆 탁자 위치 표시
4. 화살표 AR 가이드로 이동
5. 실제로 열쇠 발견

→ 기존: 평균 16분 검색
→ Space Memory: 10초 내 해결
```

#### 기능 B: 공간 메모 (Spatial Notes)

**작동 방식**:
1. 특정 위치에서 "여기에 메모" 음성 명령
2. 메모 내용을 해당 3D 좌표에 고정
3. 나중에 해당 위치 방문 시 AR로 메모 표시

**메모 유형**:
| 유형 | 예시 | 트리거 |
|-----|-----|-------|
| **알림형** | "우유 사기" (냉장고) | 위치 접근 시 팝업 |
| **정보형** | "WiFi 비번: 1234" | 터치/음성으로 확인 |
| **체크형** | "나갈 때: 지갑, 폰, 열쇠" | 현관 접근 시 체크리스트 |

**사용자 시나리오**:
```
[시나리오 2: 장보기 메모]

1. 냉장고 앞에서 우유가 없는 것 발견
2. "여기에 메모, 우유 사기"
3. 다음날 출근 전 냉장고 앞 지나감
4. AR로 "우유 사기" 메모 자동 표시
5. "확인" 누르면 메모 완료 처리

→ 기존: 까먹고 귀가 후 후회
→ Space Memory: 타이밍 맞춰 알림
```

#### 기능 C: 출발 체크리스트 (Departure Checklist)

**작동 방식**:
1. 현관/입구 위치를 "출발 존"으로 설정
2. 사용자가 출발 존에 접근하면 자동 활성화
3. 필수 물건 (지갑, 폰, 열쇠 등) 소지 여부 AR 체크
4. 미소지 물건은 마지막 위치로 안내

**사용자 시나리오**:
```
[시나리오 3: 출근 체크]

1. 현관에서 신발 신는 중
2. AR 화면에 자동 체크리스트 표시:
   ✓ 지갑 (주머니)
   ✓ 스마트폰 (손)
   ✗ 열쇠 (거실 탁자)
3. 열쇠 위치로 화살표 안내
4. 열쇠 집어들면 ✓ 업데이트
5. 모두 체크 완료 후 "좋은 하루 되세요" 메시지

→ 기존: 60%가 분실로 지각 경험
→ Space Memory: 잊는 것 방지
```

### 2.2 핵심 기능 추가: 공간 검색 (Spatial Search)

**작동 방식**:
1. "이 방에서 리모컨 어디?" 음성 질문
2. AI가 공간 스캔 + 저장된 위치 기록 + 패턴 분석
3. 확률 기반으로 가장 가능성 높은 위치 AR 표시
4. 실제 발견 시 학습하여 정확도 향상

**검색 유형**:
| 유형 | 예시 | 기술 |
|-----|-----|-----|
| **직접 검색** | "열쇠 어디?" → 마지막 위치 | 위치 기록 |
| **패턴 검색** | "보통 열쇠 어디 둬?" → 빈도 분석 | 행동 패턴 학습 |
| **추론 검색** | "리모컨 어디?" → 소파 근처 확률 | AI 상식 추론 |
| **연관 검색** | "충전기 어디?" → 콘센트 주변 | 물건 특성 추론 |

**사용자 시나리오**:
```
[시나리오 4: 공간 검색]

1. "이 방에서 리모컨 어디?"
2. AI 분석:
   - 마지막 기록: 3일 전 소파 옆
   - 자주 두는 곳: TV 앞 테이블 (60%)
   - 추론: 소파 쿠션 밑 (높은 확률)
3. AR 화면에 확률 순으로 3개 위치 표시
4. 첫 번째 위치(소파 쿠션)에서 발견
5. 시스템이 패턴 학습

→ 기존: 방 전체 뒤지기
→ Space Memory: AI가 추론해서 안내
```

**차별화 포인트**:
| Google Lens | Space Memory 공간 검색 |
|-------------|----------------------|
| "이게 뭐야?" (식별) | "어디 있어?" (위치) |
| 외부 정보 검색 | 내 공간 내 검색 |
| 일회성 | 기억 누적 + 학습 |

### 2.3 핵심 기능 추가: 3D 작업 가이드 (3D Work Guide)

**작동 방식**:
1. 작업 대상(가구, 요리 재료 등)을 카메라로 인식
2. 관련 가이드를 실물 위에 AR로 오버레이
3. 음성으로 단계 이동 (핸즈프리)
4. 실시간 오류 감지 및 피드백

**가이드 카테고리**:
| 카테고리 | 예시 | 시장 규모 | 우선순위 |
|---------|-----|---------|---------|
| **요리 가이드** | 레시피 단계별 AR | DIY $959B | **MVP** |
| **가구 조립** | IKEA, 한샘 등 | 홈 퍼니싱 대형 | MVP+1 |
| **가전 설치** | 공유기, TV 설정 | 전자제품 | MVP+1 |
| **간단한 수리** | 전구 교체, 배수구 | 홈 수리 | MVP+2 |

**사용자 시나리오**:
```
[시나리오 5: 요리 가이드]

1. "김치찌개 만들기" 시작
2. AR 화면에 첫 단계 표시:
   "돼지고기 200g 준비" + 재료 위치 표시
3. "다음" 음성 명령 → 다음 단계로
4. 타이머 필요시 AR로 남은 시간 표시
5. 완성까지 핸즈프리로 진행

→ 기존: 레시피 보고 → 손 씻고 → 다시 보고
→ Space Memory: 손 안 씻고 연속 작업
```

```
[시나리오 6: 가구 조립]

1. IKEA 책장 바코드 스캔
2. AR로 부품 목록 체크:
   ✓ 나사 A (8개)
   ✗ 육각렌치 → "서랍장 위에 있음" (물건 찾기 연동)
3. 조립 시작 → 실제 부품 위에 AR 가이드
4. "이 나사를 여기에" → 정확한 위치 표시
5. 잘못된 조립 시 경고 + 수정 가이드

→ 기존: 설명서 80% 불만족 (Design Council)
→ Space Memory: 실물 위 직접 안내
```

**기술 요구사항**:
| 기술 | 용도 | 구현 난이도 |
|-----|-----|-----------|
| Object Detection | 부품/재료 인식 | 중 |
| Pose Estimation | 조립 단계 인식 | 상 |
| 3D Model Overlay | 가이드 시각화 | 중 |
| 음성 제어 | 핸즈프리 | 하 |
| 타이머/알림 | 요리 시간 관리 | 하 |

**vs 기존 솔루션**:
| 기존 방식 | 3D 작업 가이드 |
|---------|--------------|
| 설명서 보고 → 실물 보고 | 실물 위에 바로 표시 |
| 손 멈추고 → 확인 | 핸즈프리 연속 작업 |
| 부품 찾기 → 헤매기 | AR로 정확한 부품 표시 |
| 실수 → 처음부터 | 실시간 오류 감지 |

## 3. 기술 아키텍처

### 3.1 시스템 구성

```
┌─────────────────────────────────────────────────────────────┐
│                    Space Memory 아키텍처                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [디바이스 레이어]                                          │
│   ├── iPhone (LiDAR Pro 모델) - 1차 타겟                   │
│   ├── Android (ARCore 지원) - 2차 확장                     │
│   └── AR 글래스 (향후) - 3차 확장                          │
│                                                             │
│  [AR 엔진 레이어]                                           │
│   ├── ARKit/ARCore - 공간 인식                             │
│   ├── SLAM - 실시간 매핑                                    │
│   └── Spatial Anchoring - 위치 고정                        │
│                                                             │
│  [AI 엔진 레이어]                                           │
│   ├── Object Detection (YOLO/CoreML) - 물건 인식           │
│   ├── Claude Vision - 컨텍스트 이해                        │
│   ├── Claude LLM - 자연어 처리                             │
│   └── Whisper - 음성 인식                                  │
│                                                             │
│  [데이터 레이어]                                            │
│   ├── 로컬 저장소 - 공간 맵 (프라이버시)                   │
│   ├── 클라우드 백업 - 암호화된 메모 (선택)                 │
│   └── 동기화 - 멀티 디바이스 지원                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 핵심 기술 스택

| 영역 | 기술 | 선정 이유 |
|-----|-----|---------|
| **공간 매핑** | ARKit + RoomPlan | Apple 네이티브, LiDAR 활용 |
| **물건 인식** | CoreML + YOLO | 온디바이스, 프라이버시 |
| **위치 고정** | ARKit World Map | 세션 간 위치 유지 |
| **AI 처리** | Claude 3.5 Sonnet | 컨텍스트 이해, 한국어 |
| **음성** | Whisper + TTS | 고품질 한국어 |

### 3.3 프라이버시 설계

| 원칙 | 구현 |
|-----|-----|
| **최소 수집** | 필요한 물건/위치만 저장 |
| **온디바이스** | 공간 맵은 기기 내 저장 |
| **투명성** | 저장 데이터 언제든 확인/삭제 |
| **선택적 클라우드** | 백업은 사용자 선택 |

## 5. 사용자 획득 전략

### 5.1 타겟 페르소나

#### 1차 타겟: "바쁜 김대리" (25-35세 직장인)

| 항목 | 내용 |
|-----|-----|
| **Pain Point** | 매일 아침 열쇠/지갑 찾느라 지각 위기 |
| **행동 특성** | iPhone Pro 사용, 새 앱 시도 적극적 |
| **기대 효과** | "출근 전 10분 여유 생김" |

#### 2차 타겟: "멀티태스킹 박과장" (35-45세)

| 항목 | 내용 |
|-----|-----|
| **Pain Point** | 일이 많아서 개인 물건 관리 소홀 |
| **행동 특성** | 생산성 앱 구독 의향 높음 |
| **기대 효과** | "잡생각 줄이고 업무 집중" |

### 5.2 마케팅 메시지

**헤드라인 옵션**:
