# 초기창업패키지 제출 체크리스트

> AR Agent Lab 사업계획서 제출 준비 가이드

---

## 1. 신청 자격 확인

### 1.1 기본 자격

- [ ] 창업 후 3년 이내 기업 (또는 예비창업자)
- [ ] 대표자 연령 제한 없음
- [ ] 동일 아이템 중복 지원 여부 확인

### 1.2 제외 대상 확인

- [ ] 타 정부지원사업 동시 수혜 여부
- [ ] 부정수급 이력 없음
- [ ] 국세/지방세 체납 없음

---

## 2. 평가 기준 대응 현황

### 2.1 평가 항목별 체크

| 평가 항목 | 비중 | 현재 대응 | 보완 필요 |
|---------|:----:|:-------:|:-------:|
| **창업 아이템** | 25-30% | ✅ | - |
| **문제 해결력** | 15-20% | ✅ | - |
| **시장성** | 15-20% | ✅ | - |
| **사업화 역량** | 15-20% | ⚠️ | 팀 확장 계획 구체화 |
| **성장 전략** | 10-15% | ✅ | - |
| **자금 활용** | 10% | ✅ | - |

### 2.2 항목별 대응 내용

#### 창업 아이템 (25-30%) ✅

| 요소 | 대응 내용 | 위치 |
|-----|---------|-----|
| 아이템 소개 | AR 에이전트 개인비서 서비스 | 섹션 1 |
| 개발 동기 | 시장/기술/사용자 관점 문제 인식 | 섹션 1.3 |
| 기술 차별성 | AR+AI 융합, 태그 없는 물건 추적 | 섹션 3 |
| 혁신성 | 국내 최초 B2C AR AI 비서 | 섹션 5 |

#### 문제 해결력 (15-20%) ✅

| 요소 | 대응 내용 | 위치 |
|-----|---------|-----|
| Pain Point | 연 60시간 물건 찾기, 80% 조립 불만족 | 섹션 2 |
| 타겟 고객 | 밀레니얼 직장인 (25-40세) | 섹션 2.2 |
| 솔루션 | Space Memory 5가지 핵심 기능 | 섹션 3.2 |
| 데이터 근거 | PR Newswire, Chipolo 등 출처 명시 | 전체 |

#### 시장성 (15-20%) ✅

| 요소 | 대응 내용 | 위치 |
|-----|---------|-----|
| TAM | 100조원 (공간 컴퓨팅 + 스마트 트래커 + DIY) | 섹션 4 |
| SAM | 5조원 (한국 AR 공간 서비스) | 섹션 4 |
| SOM | 60억원 (3년차, 50,000명 × ARPU 10,000원) | 섹션 4 |
| 트렌드 | AI 에이전트 CAGR 44-50%, AR CAGR 40%+ | 섹션 4 |

#### 사업화 역량 (15-20%) ⚠️

| 요소 | 대응 내용 | 보완 필요 |
|-----|---------|---------|
| 대표 역량 | IT 20년 경력 | 구체적 프로젝트 성과 추가 |
| 팀 구성 | 1인 창업 | 외주/파트너 계획 구체화 |
| 실행력 | 3개월 MVP 계획 | 진행 상황 업데이트 |

#### 성장 전략 (10-15%) ✅

| 요소 | 대응 내용 | 위치 |
|-----|---------|-----|
| 수익 모델 | 프리미엄 구독 (Free/Plus/Pro) | 섹션 6 |
| GTM 전략 | 얼리어답터 → 바이럴 → 확장 | 섹션 7 |
| 로드맵 | 3개월/6개월/12개월 마일스톤 | 섹션 10 |

#### 자금 활용 (10%) ✅

| 요소 | 대응 내용 | 위치 |
|-----|---------|-----|
| 예산 배분 | 제품 40%, 마케팅 20%, 인건비 20% 등 | 섹션 8.4 |
| 비용 상세 | Unity, Claude API, 클라우드 월별 비용 | 섹션 8.2 |
| 자금 출처 | 자기자본 + 초기창업패키지 + 후속 투자 | 섹션 8.4 |

## 4. 필수 서류 체크리스트

### 4.1 공통 서류

- [ ] 사업계획서 (양식 다운로드 후 작성)
- [ ] 대표자 신분증 사본
- [ ] 개인정보 수집/이용 동의서

### 4.2 법인/개인사업자

- [ ] 사업자등록증
- [ ] 법인등기부등본 (법인인 경우)
- [ ] 주주명부 (법인인 경우)

### 4.3 예비창업자

- [ ] 주민등록등본
- [ ] 졸업증명서 또는 재학증명서 (해당 시)

### 4.4 추가 서류 (가점)

- [ ] 특허 출원/등록 증명서
- [ ] 고객 LOI/MOU
- [ ] 프로토타입 데모 영상
- [ ] 수상 실적 증빙

---

## 5. 사업계획서 최종 검토

### 5.1 형식 검토

- [ ] 페이지 수 제한 준수 (보통 15-20페이지)
- [ ] 글자 크기/여백 양식 준수
- [ ] 오탈자 검토
- [ ] 그래프/표 가독성 확인

### 5.2 내용 검토

- [ ] 모든 수치에 출처 명시
- [ ] 경쟁사 폄하 표현 없음
- [ ] 현실적인 목표 설정
- [ ] 기술 설명보다 고객 가치 중심

### 5.3 발표 준비

- [ ] 10-15분 발표 자료 (PPT)
- [ ] 예상 질문 & 답변 준비
- [ ] 데모/프로토타입 (있을 경우)
- [ ] 1분 엘리베이터 피치
