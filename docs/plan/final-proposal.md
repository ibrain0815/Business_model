# 1. 창업 아이템 개요

## 1.1 아이템명

**AR 에이전트 개인비서 서비스 (AR Personal Agent)**

## 1.2 아이템 소개

증강현실(AR)과 AI 에이전트 기술을 결합한 차세대 개인비서 서비스입니다. 스마트글래스, 스마트폰 AR을 통해 실시간으로 사용자의 일상과 업무를 지원하는 AI 비서가 언제 어디서나 함께합니다. 일정 관리, 정보 검색, 실시간 번역, 길안내, 업무 지원 등을 AR 환경에서 직관적으로 제공합니다.

| **사용자** | 스마트폰 화면에서 벗어난 자연스러운 AI 인터랙션 니즈 |

# 3. 솔루션

## 3.1 제품/서비스 개요

**"당신 곁의 AI 비서, 눈앞에 펼쳐지다"**

AR 환경에서 작동하는 AI 에이전트 개인비서. 음성 명령과 AR 시각화를 결합하여 직관적이고 효율적인 개인비서 경험을 제공합니다.

## 3.2 핵심 기능: Space Memory (스페이스 메모리)

> "연간 60시간을 물건 찾는 데 낭비하지 마세요"

### 왜 Space Memory인가?

| 문제 | 수치 | 출처 |
|-----|-----|-----|
| 연간 분실물 검색 시간 | **2.5일 (60시간)** | PR Newswire |
| 평생 분실물 검색 시간 | **153일** | NYT |
| 분실로 인한 지각 비율 | **60%** | Chipolo |
| 미국 연간 대체 비용 | **$2.7B (3.9조원)** | PR Newswire |

### MVP 핵심 기능 5가지

| 기능 | 설명 | AR 시각화 | 차별점 |
|-----|-----|---------|--------|
| **물건 위치 기억** | AI가 물건 마지막 위치 자동 기억 | 위치 화살표 + 3D 하이라이트 | 태그 없이 무제한 추적 |
| **공간 메모** | 특정 위치에 가상 메모 고정 | 해당 위치 방문 시 AR 팝업 | 스마트폰 앱과 차별화 |
| **출발 체크리스트** | 현관에서 필수 물건 자동 체크 | 소지 여부 AR 체크리스트 | 사전 예방 알림 |
| **공간 검색** | "이 방에서 리모컨 어디?" AI 추론 | 확률 순 위치 AR 표시 | 패턴 학습 + 상식 추론 |
| **3D 작업 가이드** | 요리/조립을 실물 위 AR 가이드 | 단계별 3D 오버레이 | 핸즈프리 연속 작업 |

### 사용자 시나리오

```
[시나리오 1: 열쇠 찾기]
기존: 평균 16분 검색 → Space Memory: 10초 해결

[시나리오 2: 장보기 메모]
기존: 냉장고 확인 후 까먹음 → Space Memory: 냉장고 앞 지나갈 때 알림

[시나리오 3: 출근 체크]
기존: 60%가 분실로 지각 → Space Memory: 현관에서 자동 체크

[시나리오 4: 공간 검색]
기존: 방 전체 뒤지기 → Space Memory: AI가 "소파 쿠션 밑" 추론

[시나리오 5: 요리 가이드]
기존: 레시피 보고 → 손 씻고 → 다시 보고
→ Space Memory: 실물 위 AR + 핸즈프리 음성 제어
```

### 공간 검색: AI 추론 기반

| 검색 유형 | 예시 | 기술 |
|---------|-----|-----|

| **패턴 검색** | "보통 열쇠 어디 둬?" | 행동 패턴 학습 |
| **추론 검색** | "리모컨 어디?" → 소파 근처 확률 | AI 상식 추론 |

## 3.3 기술 아키텍처

```
┌─────────────────────────────────────────────────────────┐
│                    AR 에이전트 아키텍처                   │
├─────────────────────────────────────────────────────────┤
│  [AR 프론트엔드] - Unity Engine                          │
│   ├── Unity + AR Foundation (ARKit/ARCore)              │
│   ├── 스마트글래스 (향후 확장)                           │
│   └── 음성 인터페이스 (STT/TTS)                          │
│            ↓                                             │
│  [AI 에이전트 엔진] - Anthropic Claude                   │
│   ├── Claude 3.5 Sonnet (메인 LLM)                      │
│   ├── Claude Vision (멀티모달 처리)                      │
│   ├── 컨텍스트 인식 (위치, 시간, 상황)                    │
│   └── 개인화 엔진 (사용자 프로필, 선호도)                 │
│            ↓                                             │
│  [백엔드 서비스]                                         │
│   ├── 실시간 데이터 (날씨, 교통, 뉴스)                   │
│   ├── 지도/POI 연동 (Google Maps, 카카오맵)              │
│   └── 캘린더/메일 연동 (Google, Outlook)                 │
└─────────────────────────────────────────────────────────┘
```

## 5.4 차별화 전략

### vs AirTag/Tile (물건 찾기)
- **태그 불필요**: AI 카메라 인식으로 자동 위치 기억
- **무제한 추적**: 4개 한정 → 모든 물건
- **사전 예방**: 출발 체크리스트로 분실 방지

### vs YouTube/IKEA 설명서 (작업 가이드)
- **핸즈프리**: 손 안 씻고 연속 작업
- **실물 연동**: 설명서 해석 불필요, 실물 위 AR 표시
- **AI 지원**: "육각렌치 어디?" → 물건 찾기 연동

### vs 메모 앱 (공간 메모)
- **맥락 인식**: 냉장고 앞 지나갈 때 "우유 사기" 알림
- **3D 고정**: 해당 위치에서만 표시
- **자동 트리거**: 수동 확인 불필요

## 5.5 진입 장벽

1. **기술 융합**: AR + AI + 공간 인식 통합 (경쟁사 분리됨)
2. **데이터 Lock-in**: 공간 맵 + 물건 패턴 + 사용자 습관 축적
3. **시장 선점**: 국내 최초 "공간 기억" 서비스
4. **3개 시장 통합**: 스마트 트래커 + 메모 + 가이드 → 대체 불가
