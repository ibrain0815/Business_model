# 2025년 정부지원사업 가이드

> 작성: policy-researcher | 작성일: 2025-02-25

## 1. Executive Summary

2025년 정부 창업지원사업 총 예산은 **3조 2,940억 원**이며, AI/딥테크 분야에 대한 집중 지원이 강화되었습니다. 예비창업패키지, 초기창업패키지, 딥테크 특화 프로그램 등을 통해 AI 스타트업은 최대 2억 원까지 지원받을 수 있습니다.

---

## 2. 2025년 창업지원사업 개요

### 2.1 총 예산 규모

| 분야 | 예산 | 사업 수 |
|-----|-----|--------|
| 융자 | 1조 5,552억 원 | 12개 |
| **사업화** | **7,666억 원** | **169개** |
| 기술개발 | 6,292억 원 | 8개 |
| 시설/공간/보육 | 1,501억 원 | 123개 |
| 글로벌진출 | 1,233억 원 | 21개 |
| **합계** | **3조 2,940억 원** | - |

### 2.2 주요 변화 (2025년)

- **선택과 집중형** 창업 생태계 강화
- **딥테크 분야** 집중 지원 확대
- 추경 예산으로 AI, 바이오, 로봇 등 핵심 기술 분야 추가 지원
- 최대 **2억 원**까지 지원 가능

---

## 3. AI 스타트업 대상 주요 지원사업

### 3.1 핵심 지원 프로그램

| 프로그램 | 대상 | 지원 규모 | 특징 |
|---------|-----|----------|-----|
| **예비창업패키지** | 예비창업자 | 최대 1억 원 | 아이디어만으로 신청 가능 |
| **예비창업패키지 (딥테크)** | 예비창업자 (AI/딥테크) | 최대 2억 원 | 2025 추경 신설 |
| **초기창업패키지** | 창업 3년 이내 | 평균 7천만 원 (최대 1억 원) | 제품화~시장진입 |
| **창업도약패키지** | 창업 3-7년 | 최대 3억 원 | 스케일업 |
| **딥테크 창업 챌린지** | AI/바이오/로봇 | 최대 2억 원 | 기술 집약 스타트업 |

### 3.2 예비창업패키지 상세

**일반 트랙**:
- 대상: 사업자등록 전 예비창업자
- 지원: 최대 1억 원 (평균 4,600만 원)
- 기간: 약 10개월
- 내용: 사업화 자금, 교육, 멘토링

**딥테크 트랙 (2025 추경)**:
- 대상: AI, 바이오헬스, 미래모빌리티, 로봇, 친환경에너지 분야
- 지원: 최대 2억 원
- 특징: 핵심 기술 분야 집중 지원

### 3.3 초기창업패키지 상세

- **대상**: 창업 후 3년 이내 기업
- **지원 규모**: 평균 7천만 원 (선정평가에 따라 최대 1억 원)
- **지원 내용**: 제품화, 마케팅, 사업화 자금
- **목적**: 제품화 → 시장진입 단계 지원

### 3.4 AI 특화 프로그램

| 프로그램 | 주관 | 내용 |
|---------|-----|-----|
| 이어드림 스쿨 | KISED | AI 기술인력 양성 후 창업 연계 |
| AI 바우처 | NIPA | AI 솔루션 도입 바우처 지원 |
| 데이터 바우처 | NIA | 데이터 구매/가공 지원 |
| K-AI 제조혁신 | 중기부 | 제조업 AI 도입 지원 |

## 5. 평가 기준 분석

### 5.1 공통 평가 항목

| 평가 항목 | 비중 | 핵심 내용 |
|---------|-----|---------|
| 창업 아이템 | 25-30% | 혁신성, 차별성, 기술성 |
| 문제 해결력 | 15-20% | 시장 니즈, Pain Point 해결 |
| 시장성 | 15-20% | TAM/SAM/SOM, 성장 가능성 |
| 사업화 역량 | 15-20% | 팀 구성, 실행력, 경험 |
| 성장 전략 | 10-15% | 비즈니스 모델, 수익화 |
| 자금 활용 | 10% | 예산 계획, 집행 계획 |

### 5.2 AI 분야 가점 요소

| 가점 요소 | 내용 | 권장 사항 |
|---------|-----|---------|
| **기술 혁신성** | 특허, 논문, 기술 인증 | 특허 출원/등록 준비 |
| **시장 검증** | PoC, 파일럿, 고객 확보 | LOI/MOU 확보 |
| **팀 역량** | AI 관련 경력, 학위 | 핵심 인력 프로필 강조 |
| **사회적 가치** | 일자리 창출, ESG | 고용 계획 명시 |
| **글로벌 진출** | 해외 시장 계획 | 해외 진출 로드맵 |

### 5.3 주요 탈락 사유

1. **구체성 부족**: 막연한 계획, 수치 없는 목표
2. **차별화 미흡**: 기존 솔루션과 차이 불분명
3. **실현 가능성**: 기술/자원 대비 과도한 목표
4. **팀 역량 미검증**: 관련 경험 부족
5. **시장 분석 부실**: TAM/SAM/SOM 근거 미약

## 6. 사업계획서 작성 가이드

### 6.1 정부지원사업용 양식 구조

```
1. 창업 아이템 개요 (10%)
   └── 아이템 소개, 개발 배경, 목표

2. 문제 정의 및 솔루션 (25%)
   ├── 시장의 문제/Pain Point
   ├── 솔루션 설명
   └── 기술 차별성

3. 시장 분석 (15%)
   ├── TAM/SAM/SOM
   ├── 시장 트렌드
   └── 경쟁 분석

4. 비즈니스 모델 (15%)
   ├── 수익 모델
   ├── 가격 전략
   └── 고객 획득 전략

5. 실행 계획 (20%)
   ├── 마일스톤
   ├── 자금 활용 계획
   └── 팀 구성

6. 기대 효과 (15%)
   ├── 매출 전망
   ├── 고용 창출
   └── 사회적 가치
```

### 6.2 평가위원 관점 작성 팁

| 항목 | 팁 |
|-----|---|
| **문제 정의** | 구체적 사례와 수치로 문제의 심각성 입증 |
| **솔루션** | "왜 우리만 할 수 있는가" 명확히 |
| **시장 규모** | 신뢰할 수 있는 출처 + 계산 과정 제시 |
| **차별화** | 경쟁사 대비 표로 비교 |
| **팀** | 관련 경험/성과 구체적 수치화 |
| **재무 계획** | 현실적 + 근거 있는 추정 |

### 6.3 자주 하는 실수

1. **추상적 표현**: "혁신적", "최초" 대신 구체적 수치
2. **과도한 기술 설명**: 기술보다 고객 가치 중심
3. **비현실적 목표**: 3년 내 유니콘 → 현실적 단계별 목표
4. **시장 분석 출처 미기재**: 모든 수치에 출처 명시
5. **경쟁사 폄하**: 경쟁사 인정 + 차별점 강조

## 7. 신청 일정 및 절차

### 7.1 2025년 주요 일정

| 사업 | 공고 시기 | 신청 기간 |
|-----|---------|----------|
| 예비창업패키지 | 1-2월 | 2-3월 |
| 초기창업패키지 | 1-2월 | 2-3월 |
| 창업도약패키지 | 2-3월 | 3-4월 |
| 딥테크 추경 사업 | 추경 확정 후 | 별도 공고 |

**주의**: 상세 일정은 K-Startup에서 확인 필요

### 7.2 신청 절차

```
1. 공고 확인
   └── K-Startup, 기업마당 등에서 공고문 확인

2. 자격 검토
   └── 신청 자격, 제외 대상 확인

3. 사업계획서 작성
   └── 양식 다운로드 → 작성 → 검토

4. 온라인 신청
   └── K-Startup 회원가입 → 신청서 제출

5. 서류 평가
   └── 1차 서류 심사 (탈락률 60-70%)

6. 발표 평가
   └── 2차 대면/비대면 발표 (10-15분)

7. 선정 및 협약
   └── 최종 선정 → 협약 체결 → 사업 수행
```

## 8. 체크리스트

### 8.1 신청 전 체크리스트

- [ ] 신청 자격 확인 (업력, 대표자 조건 등)
- [ ] 중복 지원 제한 확인
- [ ] 필수 서류 준비 (사업자등록증, 주민등록등본 등)
- [ ] 사업계획서 양식 다운로드
- [ ] 신청 마감일 확인

### 8.2 사업계획서 체크리스트

- [ ] 글자 수/페이지 제한 준수
- [ ] 모든 항목 작성 완료
- [ ] 수치의 출처 명시
- [ ] 오탈자 검토
- [ ] 제3자 리뷰 완료

### 8.3 발표 평가 체크리스트

- [ ] 발표 자료 준비 (10-15분)
- [ ] 예상 질문 & 답변 준비
- [ ] 데모/프로토타입 준비 (있을 경우)
- [ ] 복장 및 태도

## 9. 결론

2025년은 AI 스타트업에게 유리한 정부지원 환경이 조성되어 있습니다. 딥테크 분야 추경 예산 확대로 최대 2억 원까지 지원 가능하며, 예비창업패키지부터 창업도약패키지까지 성장 단계별 지원 프로그램이 마련되어 있습니다.

**핵심 권고사항**:
- 딥테크 트랙 적극 활용
- 기술 차별성 + 시장 검증 강조
- 구체적 수치와 근거 제시
- 일정 미리 확인하고 여유 있게 준비
