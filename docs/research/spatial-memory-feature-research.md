### 9.5 사용자 시나리오

```
[시나리오: 공간 검색]

사용자: "이 방에서 리모컨 어디?"
시스템: (방 스캔) → "소파 쿠션 밑에 있을 확률 높음"
        → AR로 해당 위치 하이라이트
        → 실제로 발견 시 학습

차별점: 단순 위치 표시가 아닌 "확률 기반 추론"
- 마지막 본 위치
- 자주 두는 위치
- 유사 물건 패턴
```

### 9.6 AI 추론 기반 검색

| 검색 유형 | 예시 | 기술 |
|---------|-----|-----|
| **직접 검색** | "열쇠 어디?" → 마지막 위치 | 위치 기록 |
