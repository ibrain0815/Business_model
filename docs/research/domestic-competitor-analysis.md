## 2. 대기업 AI 비서 서비스

### 2.1 SKT 에이닷 (A.)

| 항목 | 내용 |
|-----|------|
| **출시** | 2023년 9월 |
| **사용자** | 550만+ (2024년 9월 기준) |
| **핵심 기능** | 통화녹음/요약, 실시간 통역, 멀티 LLM |
| **LLM 지원** | GPT-4o, Claude, 퍼플렉시티 등 8종 |
| **가격** | 무료 (기본), 프리미엄 옵션 |
| **타겟** | 일반 소비자 (B2C) |

**강점**:
- 통신사 인프라 활용 (통화 연동)
- 멀티 LLM 에이전트 지원
- 대규모 사용자 기반

**한계**:
- AR 기능 없음
- 통화 중심 기능
- 차별화된 UX 부족

### 2.2 삼성 빅스비 (Bixby)

| 항목 | 내용 |
|-----|------|
| **출시** | 2017년 (지속 업그레이드) |
| **플랫폼** | 갤럭시 스마트폰, 가전 |
| **핵심 기능** | 기기 제어, 음성 명령 |
| **AI 업그레이드** | 2024-2025년 생성형 AI 연동 예정 |
| **가격** | 무료 (기기 내장) |

**강점**:
- 삼성 기기 생태계 통합
- 하드웨어 연동 (갤럭시, 가전)
- 글로벌 사용자 기반

**한계**:
- 생성형 AI 미적용 (2024년 기준)
- 단편적 응답
- AR 기능 제한적

### 2.3 네이버 클로바X

| 항목 | 내용 |
|-----|------|
| **출시** | 2023년 (클로바X) |

| **LLM** | 하이퍼클로바X |
| **2025 업데이트** | 멀티모달, 음성 서비스 강화 |

**강점**:
- 네이버 검색/서비스 연동
- 한국어 최적화 LLM
- 네이버 생태계 (쇼핑, 지도 등)

**한계**:
- AR 기능 없음
- 네이버 서비스 종속
- 개인비서보다는 검색 중심

## 3. 국내 AI 스타트업

### 3.1 뤼튼 (Wrtn)

| 항목 | 내용 |
|-----|------|
| **설립** | 2021년 |
| **사용자** | 500만+ (MAU 600만+) |
| **투자** | 프리시리즈B 250억원 |
| **핵심 서비스** | AI 글쓰기, 이미지 생성, AI 에이전트 |
| **가격** | **무료** (핵심 차별화) |

**서비스 특징**:
- GPT-4, GPT-5 무료 무제한 제공
- AI 캐릭터 챗 서비스 '크랙' (유료)
- AI 스튜디오 (에이전트 제작)
- 한국어 최적화

**강점**:
- 무료 전략으로 대규모 사용자 확보
- 빠른 최신 AI 모델 적용
- 젊은 층 (30대 이하) 점유율 높음

**한계**:
- AR 기능 없음
- 수익 모델 제한적 (캐릭터 챗 의존)
- B2C 텍스트 중심

### 3.2 업스테이지 (Upstage)

| 항목 | 내용 |
|-----|------|
| **설립** | 2020년 |
| **누적 투자** | 2,000억원+ |
| **기업가치** | 약 7,900억원 |
| **핵심 제품** | Solar LLM, 문서 AI |
| **타겟** | B2B (기업용) |

**제품 특징**:
- Solar Pro 2: 31B 파라미터 LLM
- 문서 AI: OCR, 문서 처리
- 허깅페이스 오픈 LLM 1위 달성

**강점**:
- 자체 LLM (Solar) 보유
- 글로벌 기술력 인정 (아마존, AMD 투자)
- B2B 기업 고객 확보

**한계**:
- B2B 중심 (B2C 서비스 없음)
- AR 기능 없음
- 직접 소비자 서비스 부재

## 4. 국내 AR/XR 전문 기업

### 4.1 맥스트 (MAXST / 비트맥스)

| 항목 | 내용 |
|-----|------|
| **설립** | 2010년 |
| **사명 변경** | 비트맥스 (2025년 2월) |
| **핵심 제품** | MAXST AR SDK, MAXWORK, MAXVERSE |
| **SDK 사용** | 50개국, 28,000개 개발사, 9,000개 앱 |
| **타겟** | B2B (산업용) |

**제품**:
- **MAXST AR SDK**: AR 개발 플랫폼
- **MAXWORK**: 산업용 AR 솔루션
- **MAXVERSE**: XR 메타버스 플랫폼
- **와이더스**: 디지털 트윈 서비스 (2024)

**강점**:
- 국내 AR SDK 선두 기업
- 산업용 AR 솔루션 경험
- 글로벌 개발사 네트워크

**한계**:
- B2B 산업용 중심
- AI 비서 기능 없음
- 일반 소비자 서비스 부재

### 4.2 버넥트 (VIRNECT)

| 항목 | 내용 |
|-----|------|
| **핵심 제품** | VIRNECT Make, VIRNECT View |
| **특징** | 비코딩 AR 콘텐츠 제작 |
| **타겟** | B2B (기업용) |

**성과**:
- AR 콘텐츠 제작 기간 60일 → 5일 (12배 단축)
- AR 제작 비용 5,000만원 → 50만원 (99% 절감)

**강점**:
- 노코드 AR 제작 도구
- 기업용 AR 솔루션

**한계**:
- B2B 전용
- AI 비서 기능 없음
- 소비자 서비스 부재

## 6. 차별화 전략

### 6.1 vs 대기업 AI 비서 (에이닷, 클로바X)

| 차별화 포인트 | 대기업 | AR Agent Lab |
|-------------|-------|--------------|
| **인터페이스** | 음성/텍스트 | **AR 시각화** |
| **정보 표시** | 화면 텍스트 | **공간 오버레이** |
| **사용 환경** | 정적 | **이동 중 핸즈프리** |
| **차별 기능** | 통화 녹음 | **AR 번역, AR 내비** |

**전략**: AR이라는 새로운 인터페이스로 차별화

### 6.2 vs AI 스타트업 (뤼튼, 업스테이지)

| 차별화 포인트 | AI 스타트업 | AR Agent Lab |
|-------------|-----------|--------------|
| **형태** | 텍스트 챗봇 | **AR 비서 캐릭터** |
| **용도** | 글쓰기, 검색 | **일상 생활 지원** |
| **상호작용** | 2D 화면 | **3D AR 공간** |
| **수익모델** | 무료 (광고) | **프리미엄 구독** |

**전략**: 실용적 AR 비서로 일상 생활 침투

### 6.3 vs AR 전문 기업 (맥스트, 버넥트)

| 차별화 포인트 | AR 기업 | AR Agent Lab |
|-------------|--------|--------------|
| **타겟** | B2B (기업) | **B2C (소비자)** |
| **용도** | 산업용 | **일상 개인비서** |
| **AI** | 없음 | **Claude LLM** |
| **접근성** | 개발자 필요 | **앱 다운로드** |

**전략**: B2C 시장 선점, AI 융합으로 차별화


## 8. 결론 및 권고

### 8.1 핵심 결론

1. **시장 공백 존재**: AR + AI 비서 + B2C 영역은 국내 미개척
2. **경쟁 강도**: AI 비서는 대기업 중심, AR은 B2B 중심 → B2C AR AI는 블루오션
3. **진입 타이밍**: 대기업 AR 서비스 전 선점 기회
4. **차별화 가능**: 기술 융합 + UX 혁신으로 명확한 포지셔닝
